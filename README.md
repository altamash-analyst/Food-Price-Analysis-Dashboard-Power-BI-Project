# Food Price Analysis Dashboard | Power BI Project

A professional Business Intelligence project built using Microsoft Power BI, DAX, and Excel-based data modeling to analyze food price trends, product categories, and market insights.

This project transforms food pricing data into an interactive analytics dashboard that provides insights into product prices, categories, locations, customer ratings, and food market trends.

---

## Project Overview

The objective of this project is to develop an end-to-end data analytics solution using food price data.

This project includes:

- Data cleaning and preparation
- Data transformation
- Data modeling
- DAX measure creation
- Interactive Power BI dashboard development
- Business insight generation

The dashboard helps users analyze food price patterns, identify trends, and understand market behavior.
